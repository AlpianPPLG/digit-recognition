# MNIST Dataset Directory

This folder stores the MNIST dataset files:

- `train-images-idx3-ubyte.gz` - Training images
- `train-labels-idx1-ubyte.gz` - Training labels
- `t10k-images-idx3-ubyte.gz` - Test images
- `t10k-labels-idx1-ubyte.gz` - Test labels

The dataset will be automatically downloaded when running the training script.
